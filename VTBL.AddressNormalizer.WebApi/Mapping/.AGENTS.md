# Каталог: VTBL.AddressNormalizer.WebApi/Mapping

## Назначение
Маппинг моделей ядра в HTTP DTO (`IndoorValueDto`).

## Файлы

### IndoorValueMapper.cs
**Классы:**
- `IndoorValueMapper` (static)
  - `ToIndoorValueDto(BuildingUnitLocation)` — полный `IndoorValueDto`: все категории локации; `Name` из `CategoryNames`; `Values` = копия коллекций Location (пустые → `[]`); `null` location → пустые категории
  - `CategoryNames` — константы русских отображаемых имён категорий
