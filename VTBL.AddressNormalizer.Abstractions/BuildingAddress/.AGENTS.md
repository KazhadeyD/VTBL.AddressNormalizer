# Каталог: VTBL.AddressNormalizer.Abstractions/BuildingAddress

## Назначение
Контракты нормализации building location: extract outdoor/indoor, канонизация, end-to-end Normalize.

## Файлы

### BuildingLocationExtractionResult.cs
**Классы:**
- `BuildingLocationExtractionResult` — immutable DTO разбиения адреса
  - `Outdoor` — outdoor-часть (never null)
  - `Indoor` — indoor-часть (never null; default `""`)
  - конструктор нормализует null → `""`

### IBuildingLocationExtractor.cs
**Интерфейсы:**
- `IBuildingLocationExtractor`
  - `ExtractSplit(string input)` → `BuildingLocationExtractionResult`
  - `Extract(string input)` → outdoor-строка (`ExtractSplit().Outdoor`)

### IBuildingAddressCanonicalizer.cs
**Интерфейсы:**
- `IBuildingAddressCanonicalizer` — канонизация outdoor/building location

### IBuildingAddressNormalizer.cs
**Интерфейсы:**
- `IBuildingAddressNormalizer` — Extract + Canonicalize

### BuildingAddressNormalizationResult.cs
**Классы:**
- `BuildingAddressNormalizationResult` — результат Normalize (Original, Extracted, Canonical)

### BuildingAddressNormalizerExtensions.cs
**Классы:**
- extension-методы над `IBuildingAddressNormalizer`
