# Каталог: VTBL.AddressNormalizer.Infrastructure/BuildingAddress

## Назначение
Реализация extract/canonicalize/normalize для building location.

## Файлы

### BuildingLocationExtractor.cs
**Классы:**
- `BuildingLocationExtractor` — `IBuildingLocationExtractor`
  - `ExtractSplit(string)` — Outdoor через cutIndex + TrimTrailingDelimiters; Indoor = `Substring(indoorMatch.Index)` (не cutIndex)
  - `Extract(string)` — `ExtractSplit(input).Outdoor`
  - private helpers: `FindHouseMarkerIndex`, `ComputeCutIndex`, `TrimTrailingDelimiters`

### BuildingAddressCanonicalizer.cs
**Классы:**
- `BuildingAddressCanonicalizer` — канонизация building location

### BuildingAddressNormalizer.cs
**Классы:**
- `BuildingAddressNormalizer` — вызывает `Extract` + `ToCanonical`

### AddressPreprocessor.cs
**Классы:**
- `AddressPreprocessor` — preprocess перед extract/canonicalize

### IndoorMarkerRegistry.cs / IndoorMarkerMatch.cs
Реестр indoor-маркеров и результат поиска.

### GeographicTypeSynonymRules.cs / Steps/
Правила и шаги канонизации.
